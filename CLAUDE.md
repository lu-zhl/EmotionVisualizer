# EmotionVisualizer

An iOS application for visualizing and tracking emotions.

## Project Overview

This is an iOS application built with Swift and SwiftUI for emotion visualization and tracking.

## Technology Stack

- **Language**: Swift
- **UI Framework**: SwiftUI
- **Platform**: iOS
- **Minimum iOS Version**: iOS 17.0+

## Project Structure

```
EmotionVisualizer/
├── EmotionVisualizer/          # Main app source code
│   ├── App/                    # App entry point and configuration
│   ├── Views/                  # SwiftUI views
│   ├── Models/                 # Data models
│   ├── ViewModels/             # View models
│   └── Resources/              # Assets, colors, etc.
├── EmotionVisualizerTests/     # Unit tests
└── EmotionVisualizerUITests/   # UI tests
```

## Development Setup

1. Open the project in Xcode 15+
2. Select your target device or simulator
3. Build and run (Cmd+R)
